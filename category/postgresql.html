PostgreSQL - Tamanho do B.D. no disco

Fri 02 September 2011 | database / postgresql | Link permanente para este artigo

Para saber o tamanho de um Banco de Dados PostgreSQL no disco, execute a query sql abaixo.

$ SELECT pg_database.datname, postgres-# pg_size_pretty(pg_database_size(pg_database.datname)) AS size postgres-# FROM pg_database;

o retorno será algo no formato ...

datname | size
------------------+---------
bd-1 | 4272 kB
bd-2 | 4272 kB
bd-3 | 4364 kB
bd-4 | 4272 kB
bd-5 | 8128 kB
bd-6 | 7416 kB
bd-6 | 8336 kB
bd-7 | 4908 kB
bd-8 | 11 MB
bd-9 | 9048 kB
bd-10 | 14 MB
bd-11 | 48 MB
(12 rows)

Para um único banco de dados.

$ SELECT pg_size_pretty(pg_database_size('sorocaba_sig')) As fulldbsize;

como retorno temos ...

fulldbsize
------------
8568 kB
(1 row)

postgres=# SELECT pg_database.datname,
pg_size_pretty(pg_database_size(pg_database.datname)) AS size
FROM pg_database where datname = 'sorocaba_sig';

datname | size
--------------+---------
sorocaba_sig | 8568 kB
(1 row)

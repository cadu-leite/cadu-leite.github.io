Vagas Python Brasil

Sun 26 April 2015 Mon 20 April 2015
By Carlos Leite

Vi um post sobre categorização de uma lista de vagas Python, e decidi dar uma olhada de perto. Quem sabe não coloco algo parecido dentro do PythonPeople !?!

Não gosto muito de ver meu código depender de serviços de terceiros, principalmente quando não posso controlá-los. Mas as vezes é bom ter na manga um serviço simples que pode resolver um probleminha do dia-a-dia, seja para tomar uma decisão ou fazer alguns testes....

Acabei desconbrindo 3 serviços interessantes, sendo:

  1. Import.io
  2. Monkey Learn

  3. Plotly

    se vc conhce outros serviços parecidos, por favor me avise

o gráfico resultante pode ser visto aqui

Apesar de não ter escrito o algoritmo de classificação, fiquei empolgado pela possibilidade de poder resolver pequenos problemas de classificação sem muito esforço e de maneira rápida, na verdade sem muito código também ! ;)

Na verdade, com estes 3 serviços é possível:

  1. Fazer scrap de um site, neste caso, vagas Python no Linkedin e Exportar a lista para um CSV
  2. Categorizar cada vaga
  3. Montar um gráfico agrupando por categoria.

E tudo isso com escrevendo apenas algumas linhas de python para tratar os resultados.

Como fazer ..

  1. Vá a um site que prove uma lista.

Usei o LinkeIn para listar vagas de empregos para desenvolvedores Python (mais ou menos como no exemplo ).

Acesse a URL do Linkedin SEM LOGIN e acesse a opção jobs.

Faça a pesquisa com as palavras chaves que deseja

... deve ficar parecida com a abaixo.

https://www.linkedin.com/job/python-jobs-s%C3%A3o-paulo/?trk=jserp_search_button_execute

Copie a URL gerada no linkedin (inclusive parametros) e abra o magic.io , e coloque a url e clique em Extraxt Data

https://magic.import.io/

o magic io, é na vedade um servico, a url do site com a url de parametro deve ficar como abaixo.

https://magic.import.io/?site=https:%2F%2Fwww.linkedin.com%2Fjob%2Fpython-jobs-s%25C3%25A3o-paulo%2F%3Ftrk%3Djserp_search_button_execute

formato do CSV linkedin

columns

image - company logo (excluir: ghost_company_60x60_v1.png) title_link/text - nome do cargo company_link - no linkedin company_link/_text - nome da emprsa ou link para o serviço de emprego detailstext_value - em geral -> endereço label - em alguns (poucos) uma decricao da tarefa details_date - Data detailsblock_value - fonte com link (formato: "- From www.infojobs.com.br") abstract_description - dscricao completa.

Referências:

Post Original: MonkeyLearn Blog - "How to Create an Employment Analytics Visualization in Less Than 10 Minutes"

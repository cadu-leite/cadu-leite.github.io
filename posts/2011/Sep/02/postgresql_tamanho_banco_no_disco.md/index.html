PostgreSQL - Tamanho do B.D. no disco

Date Fri 02 September 2011 Tags python / django / REST

Para saber o tamanho de um Banco de Dados PostgreSQL no disco, execute a query sql abaixo.

postgres=# SELECT pg_database.datname,
postgres-# pg_size_pretty(pg_database_size(pg_database.datname)) AS size
postgres-# FROM pg_database;

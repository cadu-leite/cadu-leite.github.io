Tradução do Tutorial do django-rest-framewok

Date Thu 16 January 2014 Tags python / django / REST

Porque traduzir ?

Vamos usar o django-rest-framework dentro da ZNC Sistemas, empresa onde trabalho, e vamos precisar que outros desenvolvedores conheçam um pouco do framework e tenham uma idéia de como funciona. Internamente não vale a pena traduzir, o time consegue ler diocumentação em inglês, más se pensar na comunidade Django Brasil, e acho que serveria para encourajar outros a lerem o tutorial. Acredito que somos carentes de documentação em PT-BR, vai aqui uma pequena contribuição.

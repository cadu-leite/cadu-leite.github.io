Compilando Bootstrap Twitter V3 Less no Ubuntu

Date Tue 04 March 2014 | Tags web-design / less / css | Link permanente para este artigo

  1. Atualizar o node e o less, o que vem do repositório é antigo e não compila. por via das dúvidas ...

    sudo apt-get update
sudo apt-get install python-software-properties python g++ make
sudo add-apt-repository ppa:chris-lea/node.js
sudo apt-get update
sudo apt-get install nodejs

sudo apt-get autoremove node-less
sudo npm install -g less

    fonte: lessc atualizado no ubuntu e ubuntu-12-10-lessc-1-3-3-installation

  2. Baixe o o código LESS do Bootstrap-twitter, não o css compilado do site.

    O códifo fonte do bootstrap twitter é um conjunto de arquivos less, o arquivo principal a ser compilado é o bootstrap.less . Este arquivo importa todos os outros arquivos.

  3. Adicione suas customizações.

    eu preferi manter a o código original sem alterá-lo, e adicionei um arquivo theme_1.less com minhas alterações, para que fosse compilado junto . Adicionei este arquivo ao final dos imports no arquivo bootstrap.less

    ...

// Utility classes
@import "utilities.less";
@import "responsive-utilities.less";

@import "custom.less"; // << CUSTOMIZAÇÃO DO LESS

  4. Bash script para gerar um arquivo .css.

    Para testar minhas modificações mantive um bash script re-compilando o less a medida que o alterava, e gerando um arquivo .css que visualizava em um template .html

    1
2

    #!/bin/bash
[ "$2" ] || { echo "Specify both .less and .css files"; exit 1; }
inotifywait -m -e close_write . | while read x op f; do
    echo $f
    echo "Change detected. Recompiling...";
    lessc $2 > $3 && echo "`date`: COMPILED";
done

    fonte: a idéia deste script veio de um post no stack overflow

    para que o script funcione execute ...

    ./less_compile.sh  /diretorio/monitorado/  bootstrap.less  /path/custom-bootstrap.css

    ! É importante gerar o custom.css fora do path a ser monitorado, por motivos óbvios, do contrário o script bash entrará em um loop infinito de chamadas.

Uma dica após terminar a customização é utilizar o clean-css - um aplicativo node.js que minifica, formata entre outras opções o arquivo css gerado.
